1. Ранги кабуд: Методҳое, ки массиви аслиро тағир медиҳанд (Mutating methods)
Ин методҳо массиви аввалаи шуморо дигаргун мекунанд (унсур илова мекунанд, коҳиш медиҳанд ё ҷояшро иваз мекунанд).

push() — ба охири массив унсури нав илова мекунад.

JavaScript
let arr = [1, 2];
arr.push(3); // arr мешавад [1, 2, 3]
pop() — унсури охиринро аз массив нест мекунад ва онро бармегардонад.

JavaScript
let arr = [1, 2, 3];
arr.pop(); // arr мешавад [1, 2]
shift() — унсури аввалинро аз массив нест мекунад.

JavaScript
let arr = [1, 2, 3];
arr.shift(); // arr мешавад [2, 3]
unshift() — ба аввали массив унсури нав илова мекунад.

JavaScript
let arr = [2, 3];
arr.unshift(1); // arr мешавад [1, 2, 3]
splice() — методи хеле пуриқтидор. Метавонад аз дилхоҳ ҷойи массив унсурҳоро хориҷ кунад, иваз кунад ё унсури нав илова кунад.

JavaScript
let arr = [1, 5, 3];
arr.splice(1, 1, 2); // Аз индекси 1, як унсурро мегирад ва ба ҷояш 2-ро мемонад. arr мешавад [1, 2, 3]
2. Ранги зард: Методҳое, ки массиви аслиро тағир надода, массиви нав месозанд
Ин методҳо ба массиви аввала даст намезананд, балки дар асоси он як массиви нави дигар эҷод мекунанд.

concat() — ду ё якчанд массивро бо ҳам пайваст мекунад.

JavaScript
let a = [1];
let b = [2];
let c = a.concat(b); // c мешавад [1, 2] (аввалаҳо тағир намеёбанд)
slice() — як қисми массивро бурида мегирад (аз кадом индекс то кадом индекс).

JavaScript
let arr = [1, 2, 3, 4];
let newArr = arr.slice(1, 3); // аз индекси 1 то 3 (худи 3 намедарояд). newArr мешавад [2, 3]
join() — ҳамаи унсурҳои массивро ба як сатри умумӣ (string) табдил медиҳад ва онҳоро бо аломати додашуда пайваст мекунад.

JavaScript
let arr = ['A', 'B'];
let str = arr.join('-'); // str мешавад "A-B"
toString() — массивҳоро ба сатри оддӣ табдил медиҳад, ки бо вергул ҷудо шудаанд.

JavaScript
let arr = [1, 2, 3];
arr.toString(); // Натиҷа: "1,2,3"
toReversed() — ҷойи унсурҳои массивро чаппа (аз охир ба аввал) мекунад, СУРАТИ массиви навро бармегардонад (худи массиви аслӣ устувор мемонад).

JavaScript
let arr = [1, 2, 3];
let rev = arr.toReversed(); // rev мешавад [3, 2, 1]
3. Ранги сабз: Методҳое, ки танҳо як қиматро бармегардонанд (Маълумот медиҳанд)
Ин методҳо массив намесозанд, балки танҳо як қимати мушаххас (рақам ё true/false) медиҳанд.

includes() — месанҷад, ки оё унсури кофташаванда дар массив ҳаст ё не. Натиҷа true ё false мешавад.

JavaScript
let arr = [1, 2, 3];
arr.includes(2); // true бармегардонад
arr.includes(5); // false бармегардонад
indexOf() — индекс (ҷойгиршавӣ)-и унсурро дар массив меёбад. Агар унсур набошад, -1 бармегардонад.

JavaScript
let arr = ['а', 'б', 'в'];
arr.indexOf('б'); // 1 бармегардонад (чун ҳисоб аз 0 сар мешавад)
arr.indexOf('г'); // -1 бармегардонад
